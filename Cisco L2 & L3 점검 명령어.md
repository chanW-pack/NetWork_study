## L2 Switch 점검 명령어
```
terminal length 0  (터미널 길이 0으로 하여 스크립트 점검 실시)
show ver (버전 확인)
show flash: (플래시 메모리 확인)
show inventory (인벤토리 상태 확인)
sh switch detail (Stack Switch 정보 확인)
show proc cpu  (CPU 상태 확인)
show processes cpu history (시간별 CPU 상태 확인)
show proc mem (메모리 상태 확인)
show env all (팬,온도 상태 확인)
show ntp status (NTP 상태 확인)
show cdp nei (CDP 상태 확인_
show int status (인터페이스 상태 확인)
show etherchannel sum (이더채널, 포트채널 상태 확인)
show int trunk (인터페이스 트렁크 포트 확인)
show monitor session (인터페이스 미러링 상태 확인)
show ip int bri (인터페이스 IP 및 상태 확인)
show int des (인터페이스 description 상태 확인)
show vlan (Vlan 정보 확인)
show int sum (인터페이스 트래픽 확인)
show int | i error (인터페이스 에러 상태 확인)
show ip access-lists (ACL 동작 상태 확인)
show spanning-tree (스패닝 트리 프로토콜 확인)
show ip route (라우팅 테이블 확인)
show standby br (HSRP 이중화 상태 확인)
show ip eigrp topolgy sum (EIGRP 프로토콜 상태 확인)
show ip ospf nei (OSPF 프로토콜 네이버 상태 확인)
show ip bgp sum (BGP 프로토콜 상태 확인)
show ip multicast  (멀티캐스트 상태 확인)
show ip igmp groups (멀티캐스트 그룹 상태 확인)
show ip arp (ARP 테이블 확인)
show ip traffic (프로토콜 별 트래픽 상태 확인)
show run (설정 정보 확인)
show log (로그 정보 확인)
show int status err-disabled  (인터페이스 에러 포트 확인)
show clock (시간 확인)
terminal length 56 (터미널 길이 원복)
clear counters
```
